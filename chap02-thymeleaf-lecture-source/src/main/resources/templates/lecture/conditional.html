th:each

th:each 를 통해 태그를 반복할 수 있다.

이름 나이 성별 주소

th:each 에 stat 를 추가해서 반복 상태를 확인할 수 있다.

        index : 0부터 시작하는 인덱스 값
        count : 1부터 시작하는 값
        current : 현재 객체의 정보
        even : 짝수 번 째 데이터 여부
        odd : 홀수 번 째 데이터 여부
        first : 첫 번재 데이터 여부
        last : 마지막 데이터 여부
이름 나이 성별 주소 index count size current even odd first last
